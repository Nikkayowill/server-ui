ABOUT LOCALBIZ

Hi, I'm Kayo — an IT Web Programming student at NSCC in Halifax, Nova Scotia, graduating May 2026. Before diving into code, I worked as a tradesman, bouncing between odd jobs. Now I'm applying those same problem-solving skills in a different way.

I've always been an entrepreneur at heart. Before I even knew what a "backend" was, I was curious about how things worked — pre-built websites, Amazon book selling, Shopify stores, Substack's publishing platform. That curiosity eventually led me here.

WHY I BUILT THIS

A local hardware store owner approached me about setting up a web server. He wanted something reliable with that personal touch — local customer service that's seriously underrated. While we haven't reconnected recently, the conversation sparked something.

I wanted to prove I could build, deploy, and ship a real money-making application from scratch. End-to-end ownership. This project pushes my skills beyond the classroom and demonstrates I can deliver production-ready solutions.

We're not stuck in the early 2000s anymore. Physical servers, local databases, clunky setups — that's legacy thinking. Cloud hosting makes enterprise-grade infrastructure accessible and affordable for local businesses. Why sell hosting hardware when you can deliver scalable cloud solutions?

TECH STACK

Built with Node.js, Express.js, Stripe payments, deployed on DigitalOcean with PM2 process management. Security hardened with rate limiting, Helmet.js, CSRF protection, and input validation. Everything I learned in school, applied in production.

OPEN SOURCE

This project is completely open source. Transparency matters. You can view the full codebase, contribute, or learn from it here: GitHub Repository

This isn't just a school project — it's proof that I can deliver real value to real companies. If you're looking for someone who can architect, build, and ship full-stack solutions, let's talk.
